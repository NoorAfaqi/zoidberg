# Chest X-Ray Pneumonia Classification

End-to-end chest X-ray classification project for 3 classes using PyTorch and ResNet50:
- `NORMAL`
- `BACTERIAL_PNEUMONIA`
- `VIRAL_PNEUMONIA`

The repository contains:
- dataset class-separation utilities
- preprocessing pipeline code
- model training notebook
- Streamlit inference app

## Project Structure

- `Class_Separation.ipynb`: splits original `PNEUMONIA` images into bacterial vs viral class folders using filename keywords
- `Preprocessing_Pipeline.ipynb`: builds datasets/loaders and visual checks for preprocessing
- `preprocessing.py`: reusable preprocessing transforms module used by inference
- `Resnet50_Training.ipynb`: trains and evaluates ResNet50, then exports weights/classes
- `app.py`: Streamlit app for single-image inference
- `classes.json`: class labels used by the app
- `resnet50_pneumonia.pth`: trained model weights
- `requirements.txt`: Python dependencies

## Dataset

Source dataset: [Chest X-Ray Images (Pneumonia)](https://www.kaggle.com/datasets/paultimothymooney/chest-xray-pneumonia)

Expected path:

```text
datasets/chest_Xray/
  train/
    NORMAL/
    BACTERIAL_PNEUMONIA/
    VIRAL_PNEUMONIA/
  val/
    NORMAL/
    BACTERIAL_PNEUMONIA/
    VIRAL_PNEUMONIA/
  test/
    NORMAL/
    BACTERIAL_PNEUMONIA/
    VIRAL_PNEUMONIA/
```

Notes:
- the original Kaggle structure has a combined `PNEUMONIA` folder
- run `Class_Separation.ipynb` first to split it into bacterial/viral classes

## Preprocessing Pipeline

Image preprocessing implemented in `preprocessing.py` and used in notebooks/app:
- grayscale-to-RGB conversion for compatibility with ImageNet-pretrained models
- CLAHE enhancement (`clipLimit=2.0`, `tileGridSize=(8, 8)`)
- Gaussian blur denoising (`3x3` kernel)
- resize to `224x224`
- tensor conversion and normalization with
  - mean: `[0.485, 0.485, 0.485]`
  - std: `[0.229, 0.229, 0.229]`

Training augmentation (`train_transforms`) additionally applies:
- random horizontal flip
- random rotation (`10` degrees)
- random resized crop (`scale=(0.9, 1.1)`)

Validation/test/inference uses `val_test_transforms` without random augmentation.

## Training

Training is in `Resnet50_Training.ipynb`:
- model: `torchvision.models.resnet50` (pretrained ImageNet backbone)
- final layer adapted to number of project classes
- loss: cross-entropy
- optimizer: Adam (`lr=1e-4`)
- epochs: `10` (current notebook setting)
- device: CUDA if available, else CPU

Outputs generated by training:
- `resnet50_pneumonia.pth` (model state dict)
- `classes.json` (class index-to-label list)

## Inference App (Streamlit)

`app.py` loads:
- `resnet50_pneumonia.pth`
- `classes.json`
- `val_test_transforms` from `preprocessing.py`

Features:
- upload `jpg/jpeg/png` chest X-ray
- class prediction
- confidence bar
- top-3 class probabilities

Run locally:

```bash
streamlit run app.py
```

## Installation

Create and activate a virtual environment, then install dependencies:

```bash
pip install -r requirements.txt
```

Main dependencies:
- `torch`, `torchvision`, `torchaudio`
- `opencv-python`
- `numpy`
- `pillow`
- `streamlit`

## Recommended Workflow

1. Place dataset under `datasets/chest_Xray/`
2. Run `Class_Separation.ipynb`
3. Run `Preprocessing_Pipeline.ipynb` to validate transforms/loaders
4. Train model via `Resnet50_Training.ipynb`
5. Start `streamlit run app.py` for inference
